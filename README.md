<h1>Система формирования заказов на стройматериал</h1>
<p>Реализация в виде консольного приложения системы формирования заявки на закупку конкретно одного наименования строительного материала с элементами маркетингового удержания клиента.</p>
<h2>О программе</h2>
Приложение выполняет последовательность действий:

Пользователь сначала выбирает регион, потом ему открывается список доступных в регионе товаров по ценам данного региона. Пользователь выбирает товар, система у него спрашивает: хочет ли он оформить заявку.
<li>Если соглашается - происходит оформление</li>

<li><p>Если пользователь отказывается от оформления заказа, то программа пробует его удержать: предлагает либо самую дешёвую доступную алтернативу в заданном регион, а если текущий товар уже самый дешеёвый - делает на него скидку в 5%. Если аналогов товара в регионе нет, то программа принимает отказ</p>
<p>Если пользователь согласен на замену, то оформляется заявка</p></li>

<h2>Запуск</h2>
<p>Windows PS: <code>путь_до_репо\BuildingMaterialsSingle> node .\src\app.js</code></p>
<p>Linux: <code>путь_до_репо/BuildingMaterialsSingle ~$ node ./src/app.js</code></p>
